1. POST/api/1/item - создать объявление
   
   1. Создание объявления с валидными данными
      Шаги:
      1. Отправить запрос POST/api/1/item с телом запроса:
         {
          "sellerID": 123456,
          "name": "Phone",
          "price": 10000,
          "statistics": {
            "likes": 0,
            "viewCount": 0,
            "contacts": 0
            }
          }
      Ожидаемый результат: статус 200, ответ содержит поля id, sellerId, name, price, statistics, createdAt

  2. Создание объявления - отсутствует sellerID
     Шаги: Отправить POST/api/1/item с отсутствующим полем "sellerID"
     Ожидаемый результат: статус 400, тело содержит объект result.messages

  3. Создание объявления - sellerID неверного типа
     Шаги: Отправить POST/api/1/item, где "sellerID": "string"
     Ожидаемый результат: статус 400, тело содержит объект result.messages

  4. Создание объявления - отсутствует name
     Шаги: Отправить POST/api/1/item с отсутствующим полем "name"
     Ожидаемый результат: статус 400, тело содержит объект result.messages

  5. Создание объявления - отсутствует price
     Шаги: Отправить POST/api/1/item с отсутствующим полем "price"
     Ожидаемый результат: статус 400, тело содержит объект result.messages

  6. Создание объявления - statistics неверной структуры
     Шаги: Отправить POST/api/1/item, где "statistics": "string"
     Ожидаемый результат: статус 400, тело содержит объект result.messages

  7. Ошибка сервера
     Шаги: Смоделировать ситуацию неверного формата JSON
     Ожидаемый результат: статус 500, тело содержит объект result.messages

2. GET/api/1/item/{id} - получить объявление по id

   1. Получение объявления по существующему id
      Шаги: Отправить GET/api/1/item/{validId}
      Ожидаемый результат: статус 200, тело: массив объявлений согласно схеме

   2. ID не существует
      Шаги: Отправить GET/api/1/item/{invalidId}
      Ожидаемый результат: статус 404, тело содержит объекты result и status

   3. Неверный формат id
      Шаги: Отправить GET/api/1/item/str
      Ожидаемый результат: статус 400, тело содержит объект result.messages

   4. Ошибка сервера
      Шаги: Отправить слишком длинный id
      Ожидаемый результат: статус 500, тело содержит объект result.messages
